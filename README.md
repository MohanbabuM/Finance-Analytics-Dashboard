# Finance Analytics Dashboard | Power BI

## Project Overview

Developed an interactive Finance Analytics Dashboard in Power BI to monitor financial transactions, customer behavior, operational fees, taxes, and overall business performance. The solution provides real-time insights through dynamic KPIs, drill-down capabilities, and advanced visualizations to support data-driven decision-making.

## Business Objectives

* Track transaction growth and financial performance.
* Analyze monthly transaction trends and YoY changes.
* Monitor transaction success, failure, and pending rates.
* Identify high-performing customer segments and regions.
* Evaluate transaction profitability through fees and taxes.
* Understand customer demographics and transaction patterns.

## Key KPIs

* Total Transaction Amount
* Total Transactions
* Average Transaction Value
* Total Fees Collected
* Total Tax Generated
* Year-over-Year (YoY) Growth Analysis

## Dashboard Features

* Monthly Transaction Trend Analysis
* Transaction Status Distribution (Success, Failed, Pending)
* Customer Segment Performance Analysis
* State-wise Financial Performance Tracking
* Transaction Type Profitability Matrix
* Gender-based Customer Analysis
* Interactive Drill-through and Detailed Record View
* Dynamic Filters for Year, Occupation, Category, and KPI Selection

## Tools & Technologies

* Power BI Desktop
* Power Query
* DAX
* Data Modeling
* Interactive Visualizations

## Business Impact

Enabled stakeholders to monitor financial KPIs in real time, identify growth opportunities, optimize transaction performance, and make informed strategic decisions through a centralized analytics platform.
